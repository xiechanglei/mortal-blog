# Mortal's Blog

25年接近年尾一个阳光明媚的早晨，刚到公司，离上班还有一个小时，手里捧着一杯热乎乎的白开水，旁边是刚买的感冒药，最近流感袭来，不知从前几年的什么时候开始，身体好像变的很脆弱，气温骤降下，稍不注意就感冒。

一直有做个轻博客的想法，偶尔记录一下生活，当然与工作无关，因为工作中的技术性文章，如果我要去写的话，一个是需要查阅大量的资料，因为技术是严谨的，另外一个是可能需要持续的跟进，我也没有那么多的时间和精力去维护一个技术博客，所以就想写一些生活中的琐事，记录一下自己的心情。

最近有看到一篇使用cloudflare部署网站的一些文章，觉得挺有意思的，可以尝试做一下，整体的思路是使用markdown文件来编写文章，然后使用nodejs生成静态网站，最后部署到cloudflare上面去。

> 开始是考虑使用vercel的，但是后来发现vercel国内访问好像有问题，解决的话有些麻烦，所以就换成cloudflare了。

试试吧， 希望未来的某一天，翻阅这些文字的时候，能回忆起当时的心情和故事。

以下是一些待完成的功能：

- 友站的页面
- 文章搜索分类，文章列表顶部做个分类的tags切换
- 静态资源看看做个cdn加速
- 美化滚动条
